pulsesuite.core.typetpamedium.writetpaparams(cmd: str | pathlib.Path, tpa: tpas) → None¶
Write all TPA parameters in a compact binary form:

[conc, levels, r, sigspa, sigIspa, sigtpa, sigItpa]

Arrays are dumped in C-order; only the first levels`×`levels (or levels) entries are stored.

pulsesuite.core.typetpamedium.readtpaparams(cmd: str | pathlib.Path, tpa: tpas) → None¶

Read back the binary format written by writetpaparams. Overwrites the fields in the given tpas in-place.
